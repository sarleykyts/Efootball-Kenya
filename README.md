# Efootball-Kenya
eFootball Kenya Tournament Platform
A modern, mobile-first Progressive Web App (PWA) for managing eFootball tournaments in Kenya.

## Features
- Multiple tournament formats
- Player registration with payment confirmation
- Live scoring + screenshot proof
- Score verification & dispute resolution
- Attendance tracking
- Player reputation system
- Achievement badges
- Animated Trophy Cabinet
- Fantasy eFootball League + Team Builder
- Prediction League
- Live stream embedding + chat
- Admin dashboard with analytics
- Glassmorphism UI with smooth animations

## Deployment
Hosted on GitHub Pages.
